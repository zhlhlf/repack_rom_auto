## 如何使用
- 在WSL、ubuntu、deepin等Linux下
```shell
    sudo apt update
    sudo apt upgrade
    sudo apt install git -y
    # 克隆项目
    git clone https://github.com/zhlhlf/repack_rom_auto
    cd repack_rom_auto
    # 安装依赖
    sudo ./bin/setup.sh
    # 开始移植
    sudo ./bin/start.sh <底包路径> <移植包路径>
```
